# 🧠 Quiz Game (Python CLI)

Simple True/False quiz game built using Python.

---

## 🚀 Features
- True/False question system  
- Tracks score after each question  
- User input-based CLI game  
- Real-time answer validation  
- Final score display  

---

## 🛠️ Tech Stack
- Python 3  
- Object-Oriented Programming (OOP)  
- Command Line Interface (CLI)  

---

## 📁 Project Structure
- `main.py` → Runs the quiz  
- `model.py` → Stores questions  
- `quiz_brain.py` → Quiz logic  

---

## ▶️ How to Run
```bash
python main.py

##🖥️ Gameplay
- Answer using: True or False
- Score updates after each question
- Final result shown at the end
