🧠 Quiz Game (Python CLI)
🎯 Simple True/False quiz game built using Python
📚 Uses Object-Oriented Programming (OOP) concepts
🔄 Iterates through multiple questions and tracks score
💡 User input-based interactive command line game
🚀 Features
✅ Question bank loaded from structured data
✅ OOP with separate classes (Question, QuizBrain)
✅ Real-time answer checking and scoring
✅ Clean game loop implementation
✅ Final score displayed at the end
🛠️ Tech Stack
🐍 Python 3
🧩 OOP Concepts
💻 CLI (Command Line Interface)
📂 Project Structure
├── main.py          # Runs the quiz game
├── data.py          # Contains question data
├── question_model.py# Question class
├── quiz_brain.py    # Quiz logic & scoring
▶️ How to Run
python main.py
🎮 Gameplay
Answer questions with: True or False
Score updates after each question
Final result shown at the end
📌 Note
Question data is static (can be extended easily)
Based on quiz-style logic (similar to trivia apps)
